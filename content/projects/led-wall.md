Homemade game console
{{< /lead >}}

I made a homemade game console from a 32x64 LED panel. It runs [guideline](https://tetris.wiki/Tetris_Guideline)-compliant Tetris, [Conway's Game of Life](https://en.wikipedia.org/wiki/Conway%27s_Game_of_Life), and (of course) a 4D Rubik's Cube simulator inspired by [Flat Hypercube](https://github.com/milojacquet/flat-hypercube).

<!--more-->

The source code can be found on [GitHub](https://github.com/HactarCE/ledwall).
